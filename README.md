## 我做过的方向

### 宇树机器人 Agent 智能导览系统

- 基于 `Spring Boot + LangChain4j + RAGFlow` 搭建导览 Agent 后端链路。
- 支持 `知识入库 -> 检索召回 -> Prompt 组装 -> Tool Calling -> 流式输出 -> 语音播报` 的完整闭环。
- 参与机器人鉴权、地图点位、导览路线、场景模板、双语讲解语料和管理后台等能力建设。

### Rokid 公安眼镜智能识别中间件

- 基于 `Spring Boot 3 + Java 17` 封装 Rokid 眼镜与海康 Artemis 平台之间的统一中间件接口。
- 实现设备管理、人脸库管理、设备与库绑定、识别调用和结果封装返回。
- 负责项目 `公安现场部署、配置适配、接口联调与内网上线交付`。

### 工程化与交付

- 熟悉 `MySQL / Redis / RabbitMQ / WebSocket / SSE / Docker / Linux` 等常见工程能力。
- 负责过公司线上服务器维护、版本发布、配置调整和上线后问题排查。
- 关注的不只是“功能做出来”，而是“系统能不能稳定交付、稳定运行”。
